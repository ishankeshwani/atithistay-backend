# AtithiStay – Hotel Booking Backend

AtithiStay is a Spring Boot backend application inspired by Airbnb and OYO.

## Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA
- PostgreSQL

## Features
- User registration
- Hotel management
- Room management
- Booking with date overlap prevention
- Booking cancellation

## Architecture
Controller → Service → Repository → Database
